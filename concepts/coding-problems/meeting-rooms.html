Meeting Rooms

Can you attend every meeting? Only if none overlap. Sort by start and check each meeting begins after the previous ends.

#intervals #sorting #easy #overlap #sweep #interview

The problem

Given meeting time intervals, return true if a single person can attend all of them (no two overlap).

📥 Example

[[0,30],[5,10],[15,20]] → false  ·  [[7,10],[2,4]] → true

The idea

Sort the meetings by start time. Then any conflict must be between adjacent meetings: if a meeting starts before the previous one ends, they overlap and the answer is false. One linear scan after sorting settles it.

Time O(n log n), Space O(1).

C# solution

C# 
public bool CanAttendMeetings(int[][] intervals)
{
    Array.Sort(intervals, (a, b) => a[0] - b[0]);   // sort by start
    for (int i = 1; i < intervals.Length; i++)
        if (intervals[i][0] < intervals[i - 1][1])   // starts before prev ends
            return false;
    return true;
}

Complexity

⏱️ Complexity

  • Time: O(n log n) — sort then a linear scan.
  • Space: O(1) beyond sorting.

Pattern & takeaways

🎯 Remember these three

  • Sort by start; conflicts can only be between adjacent meetings.
  • Overlap test: start[i] < end[i−1].
  • It’s the feasibility check underneath Meeting Rooms II.
