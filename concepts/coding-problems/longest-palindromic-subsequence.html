Longest Palindromic Subsequence

Longest subsequence that reads the same both ways. An interval DP: matching ends add 2 to the inner interval; mismatches drop one end.

#dynamic-programming #2d-dp #strings #medium #interval-dp #interview

The problem

Given a string s, return the length of its longest palindromic subsequence (characters in order, not necessarily contiguous).

📥 Example

"bbbab" → 4 ("bbbb")  ·  "cbbd" → 2 ("bb")

Brute force

Equivalent to the LCS of s and its reverse, but the direct interval DP is cleaner. Brute-force checking all subsequences is exponential.

Optimized approach

Let dp[i][j] = longest palindromic subsequence within s[i..j]. If s[i] == s[j], they extend the best inner palindrome: dp[i][j] = 2 + dp[i+1][j−1] (or 1 if i == j). If they differ, drop one end: dp[i][j] = max(dp[i+1][j], dp[i][j−1]). Single characters are length-1 palindromes. Fill by increasing interval length; the answer is dp[0][n−1].

Time O(n²), Space O(n²).

C# solution

C# 
public int LongestPalindromeSubseq(string s)
{
    int n = s.Length;
    var dp = new int[n, n];
    for (int i = n - 1; i >= 0; i--)
    {
        dp[i, i] = 1;                              // single char
        for (int j = i + 1; j < n; j++)
            dp[i, j] = s[i] == s[j]
                ? dp[i + 1, j - 1] + 2
                : Math.Max(dp[i + 1, j], dp[i, j - 1]);
    }
    return dp[0, n - 1];
}

Complexity

⏱️ Complexity

  • Time: O(n²).
  • Space: O(n²) (reducible to O(n)).

Pattern & takeaways

🎯 Remember these three

  • Interval DP: solve substrings by increasing length.
  • Matching ends add 2 to the inner interval; else drop one end.
  • It equals LCS of s and reverse(s).
