Leaf-Similar Trees

Read each tree's leaves left to right; the trees are leaf-similar exactly when those two sequences match.

#trees #dfs #easy

The problem

Two trees are leaf-similar if their leaves, in left-to-right order, form the same sequence. Return whether two given trees are leaf-similar.

📥 Example

Both trees yielding leaves [6,7,4,9,8] → true

Brute force

There's no avoiding visiting the leaves; the work is collecting them in order and comparing.

Optimized approach

DFS each tree, appending a value whenever a node is a leaf (no children). Compare the two resulting lists element by element.

Time O(n), Space O(n).

C# solution

C# 
public bool LeafSimilar(TreeNode root1, TreeNode root2)
{
    var a = new List<int>();
    var b = new List<int>();
    void Leaves(TreeNode node, List<int> acc)
    {
        if (node == null) return;
        if (node.left == null && node.right == null) { acc.Add(node.val); return; }
        Leaves(node.left, acc);
        Leaves(node.right, acc);
    }
    Leaves(root1, a);
    Leaves(root2, b);
    return a.SequenceEqual(b);
}

Complexity

⏱️ Complexity

  • Time: O(n) — visit every node of both trees.
  • Space: O(n) — the two leaf lists.

Pattern & takeaways

🎯 Remember these three

  • A left-to-right DFS collects leaves in the required order.
  • Compare the sequences with SequenceEqual.
  • Only leaves (no children) are recorded — internal nodes are skipped.
