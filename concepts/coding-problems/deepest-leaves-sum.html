Deepest Leaves Sum

The deepest leaves are exactly the last level of a BFS, so sum that level and you are done.

#trees #bfs #medium

The problem

Given a binary tree, return the sum of the values of its deepest leaves.

📥 Example

Tree [1,2,3,4,5,null,6,7,null,null,null,null,8] → 15 (7 + 8)

Brute force

A depth-tracking DFS can accumulate per-depth sums, but BFS is the most direct: the final level is precisely the deepest leaves.

Optimized approach

BFS level by level. Keep a running sum for the current level, resetting it at the start of each level. When the queue empties, the last level's sum is the answer.

Time O(n), Space O(w).

C# solution

C# 
public int DeepestLeavesSum(TreeNode root)
{
    var q = new Queue<TreeNode>();
    q.Enqueue(root);
    int sum = 0;
    while (q.Count > 0)
    {
        int size = q.Count;
        sum = 0;                       // reset for this level
        for (int i = 0; i < size; i++)
        {
            var node = q.Dequeue();
            sum += node.val;
            if (node.left != null) q.Enqueue(node.left);
            if (node.right != null) q.Enqueue(node.right);
        }
    }
    return sum;
}

Complexity

⏱️ Complexity

  • Time: O(n) — each node enqueued once.
  • Space: O(w) — w = widest level.

Pattern & takeaways

🎯 Remember these three

  • The deepest leaves are the last BFS level — sum it.
  • Reset the running sum at the start of each level.
  • Same level-loop template as right-side view and level averages.
