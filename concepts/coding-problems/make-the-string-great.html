Make The String Great

A stack cancels bad pairs as they form: if the incoming char is the same letter as the top but opposite case, they annihilate.

#stack #strings #easy

The problem

Repeatedly remove adjacent characters that are the same letter in different cases (e.g. 'a' and 'A') until none remain, then return the string.

📥 Example

"leEeetcode" → "leetcode"

Brute force

Rescanning after each removal is O(n²). A stack resolves all cancellations in one pass.

Optimized approach

Scan with a stack. If the top and current characters are the same letter but differ in case (their difference is 32, or they're equal under ToLower yet not identical), pop; otherwise push. The stack is the result.

Time O(n), Space O(n).

C# solution

C# 
public string MakeGood(string s)
{
    var stack = new Stack<char>();
    foreach (char c in s)
    {
        if (stack.Count > 0 && stack.Peek() != c &&
            char.ToLower(stack.Peek()) == char.ToLower(c))
            stack.Pop();
        else
            stack.Push(c);
    }
    var arr = stack.ToArray();
    Array.Reverse(arr);
    return new string(arr);
}

Complexity

⏱️ Complexity

  • Time: O(n) — each character pushed/popped once.
  • Space: O(n) — the stack.

Pattern & takeaways

🎯 Remember these three

  • Same-letter, opposite-case ⇔ differ by 32, or equal under ToLower but not identical.
  • A stack cancels pairs in a single pass.
  • Reverse the stack to rebuild the string.
