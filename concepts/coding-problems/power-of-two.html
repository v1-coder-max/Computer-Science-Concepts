Power of Two

A power of two has exactly one bit set. The trick n & (n−1) clears the lowest set bit, so for a true power of two it leaves zero.

#bit-manipulation #math #easy

The problem

Given an integer n, return true if it is a power of two (1, 2, 4, 8, …).

📥 Example

16 → true (10000₂); 6 → false

Brute force

Repeatedly dividing by 2 while even and checking you end at 1 is O(log n). The bit trick answers it in O(1).

Optimized approach

A power of two in binary is a single 1 followed by zeros. The operation n & (n − 1) clears the lowest set bit; if the result is 0 (and n > 0), there was exactly one set bit — a power of two.

Time O(1), Space O(1).

C# solution

C# 
public bool IsPowerOfTwo(int n)
{
    return n > 0 && (n & (n - 1)) == 0;
}

Complexity

⏱️ Complexity

  • Time: O(1) — a single bitwise operation.
  • Space: O(1) — constant.

Pattern & takeaways

🎯 Remember these three

  • A power of two has exactly one set bit.
  • n & (n − 1) clears the lowest set bit — zero means a single bit.
  • Guard n > 0: zero and negatives are not powers of two.

Interview tip

🗣️ When this comes up

The one-liner is impressive only if you explain why n & (n−1) works and remember the n > 0 guard. The same trick underlies counting set bits (Brian Kernighan's algorithm).
