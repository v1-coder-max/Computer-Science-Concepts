Surrounded Regions

Flip every O region enclosed by X. The shortcut: an O survives only if it connects to the border — so mark border-reachable O’s first.

#graphs #dfs #grid #medium #border-flood #interview

The problem

Given a board of X and O, capture all regions of O that are completely surrounded by X (flip those Os to X). Border-connected Os are not captured.

📥 Example

A board where interior Os become X but edge-connected Os stay.

Brute force

For each O, flood-fill to check whether its region touches a border — O((m·n)²) with repeated work across the same region. Flooding from the borders once flips it to O(m·n).

Optimized approach

Invert the logic: an O is safe iff it’s reachable from a border O. DFS/BFS from every O on the four edges, marking reached cells with a temporary sentinel (e.g. #). After that, every remaining O is enclosed → flip it to X, and restore the sentinels back to O.

Time O(m·n), Space O(m·n).

C# solution

C# 
public void Solve(char[][] board)
{
    int m = board.Length, n = board[0].Length;
    void Dfs(int r, int c)
    {
        if (r < 0 || c < 0 || r >= m || c >= n || board[r][c] != 'O') return;
        board[r][c] = '#';                       // mark safe
        Dfs(r+1,c); Dfs(r-1,c); Dfs(r,c+1); Dfs(r,c-1);
    }
    for (int r = 0; r < m; r++) { Dfs(r, 0); Dfs(r, n - 1); }
    for (int c = 0; c < n; c++) { Dfs(0, c); Dfs(m - 1, c); }

    for (int r = 0; r < m; r++)
        for (int c = 0; c < n; c++)
            board[r][c] = board[r][c] == '#' ? 'O' : 'X';   // restore / capture
}

Complexity

⏱️ Complexity

  • Time: O(m·n) — one border flood plus one sweep.
  • Space: O(m·n) recursion worst case.

Pattern & takeaways

🎯 Remember these three

  • Invert: find the safe (border-connected) O’s, capture the rest.
  • A temporary sentinel marks safe cells during the flood.
  • A final sweep flips enclosed O→X and restores sentinels.
