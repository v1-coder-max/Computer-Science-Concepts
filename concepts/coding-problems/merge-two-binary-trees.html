Merge Two Binary Trees

Lay one tree over the other: where both have a node, add the values; where only one exists, keep it.

#trees #dfs #recursion #easy

The problem

Given two binary trees, merge them so overlapping nodes are summed and non-overlapping nodes are carried over. Return the merged tree.

📥 Example

Roots 1 and 2 overlap → 3; a node present in only one tree is kept as-is.

Brute force

There's no shortcut around visiting the union of nodes; a single simultaneous recursion does exactly that.

Optimized approach

Recurse on both trees together. If one node is null, return the other. Otherwise create a node with the summed value and recurse on the left pair and the right pair. (You can also merge into the first tree in place.)

Time O(n), Space O(h).

C# solution

C# 
public TreeNode MergeTrees(TreeNode t1, TreeNode t2)
{
    if (t1 == null) return t2;
    if (t2 == null) return t1;
    var node = new TreeNode(t1.val + t2.val);
    node.left = MergeTrees(t1.left, t2.left);
    node.right = MergeTrees(t1.right, t2.right);
    return node;
}

Complexity

⏱️ Complexity

  • Time: O(n) — n = nodes in the smaller overlap.
  • Space: O(h) — recursion = height.

Pattern & takeaways

🎯 Remember these three

  • Recurse on both trees in lockstep, summing where they overlap.
  • A null on either side short-circuits to the other subtree.
  • Merging in place saves the new allocations if mutation is allowed.
