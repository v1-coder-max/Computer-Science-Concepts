Add Digits

Summing digits over and over collapses any number to a single digit. That digital root also has a neat closed form using mod 9.

#math #easy

The problem

Repeatedly replace a non-negative integer by the sum of its digits until a single digit remains; return it.

📥 Example

38 → 3+8=11 → 1+1=2 → 2

Brute force

Looping the digit-sum until a single digit is fine and runs in a handful of iterations. The math identity does it in O(1).

Optimized approach

The digital root satisfies dr(n) = 1 + (n − 1) mod 9 for n > 0 (and 0 for 0), because summing digits preserves value mod 9. That collapses the whole process to one expression.

Time O(1), Space O(1).

C# solution

C# 
public int AddDigits(int num)
{
    return num == 0 ? 0 : 1 + (num - 1) % 9;
}

Complexity

⏱️ Complexity

  • Time: O(1) — a single arithmetic expression.
  • Space: O(1) — constant.

Pattern & takeaways

🎯 Remember these three

  • Digit-summing preserves a number's value mod 9.
  • Digital root = 1 + (n − 1) mod 9 (and 0 for 0).
  • Know both: the intuitive loop and the O(1) formula.
