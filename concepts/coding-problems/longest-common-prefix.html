Longest Common Prefix

The longest prefix shared by all strings. Vertical scanning: compare character position 0 across all strings, then 1, and stop at the first mismatch.

#strings #easy #scanning #prefix #interview

The problem

Given an array of strings strs, return the longest common prefix; return "" if there is none.

📥 Example

["flower","flow","flight"] → "fl"  ·  ["dog","racecar","car"] → ""

Brute force

Compare strings pairwise, shrinking a running prefix — works but re-scans repeatedly. Vertical scanning compares one column at a time and stops early at the first divergence.

Optimized approach

Take the first string as a reference. For each character index i, check that every other string has the same character at i (and is long enough). The moment a string is too short or differs, return the prefix up to i. If you exhaust the reference string, it’s entirely a common prefix.

Time O(total characters), Space O(1).

C# solution

C# 
public string LongestCommonPrefix(string[] strs)
{
    if (strs.Length == 0) return "";
    for (int i = 0; i < strs[0].Length; i++)
    {
        char c = strs[0][i];
        for (int s = 1; s < strs.Length; s++)
            if (i >= strs[s].Length || strs[s][i] != c)
                return strs[0].Substring(0, i);     // first divergence
    }
    return strs[0];                                 // first string is the prefix
}

Complexity

⏱️ Complexity

  • Time: O(total characters) worst case.
  • Space: O(1).

Pattern & takeaways

🎯 Remember these three

  • Vertical scan compares a column across all strings and stops early.
  • Guard against strings shorter than the current index.
  • Sorting + comparing first/last string is a neat alternative.
