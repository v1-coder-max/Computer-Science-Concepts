Remove Duplicates from Sorted List

Strip duplicates from a sorted list so each value appears once. One pass — skip over nodes equal to the current one.

#linked-list #two-pointers #easy #in-place #sorted #interview

The problem

Given the head of a sorted linked list, delete all duplicates so that each value appears only once, and return the list.

📥 Example

1→1→2 → 1→2  ·  1→1→2→3→3 → 1→2→3

Brute force

A HashSet of seen values works but is O(n) space and ignores the sorted order. Because duplicates are adjacent in a sorted list, a single in-place pass suffices.

Optimized approach

Walk with a current pointer. While current.next has the same value as current, relink current.next = current.next.next to skip the duplicate (don’t advance). When the next value differs, advance current. One pass keeps the first occurrence of each value.

Time O(n), Space O(1).

C# solution

C# 
public ListNode DeleteDuplicates(ListNode head)
{
    ListNode current = head;
    while (current != null && current.next != null)
    {
        if (current.next.val == current.val)
            current.next = current.next.next;     // skip the duplicate
        else
            current = current.next;               // move on
    }
    return head;
}

Complexity

⏱️ Complexity

  • Time: O(n) — one pass.
  • Space: O(1).

Pattern & takeaways

🎯 Remember these three

  • Sorted list ⇒ duplicates are adjacent, so skip in place.
  • Don’t advance current while it keeps matching.
  • The “remove all numbers that ever duplicate” variant needs a dummy head.

Interview tip

🗣️ When this comes up

Sortedness makes it a clean in-place skip. Note the harder variant (delete all nodes that have duplicates) needs a dummy head and a different loop — mention you know the distinction.
