Interleaving String

Is s3 a perfect interleaving of s1 and s2? A 2-D reachability DP over “how many chars of each have we consumed.”

#dynamic-programming #2d-dp #strings #medium #reachability #interview

The problem

Given s1, s2, and s3, return true if s3 can be formed by interleaving s1 and s2 while preserving each one’s order.

📥 Example

s1="aabcc", s2="dbbca", s3="aadbbcbcac" → true

Brute force

Recursively, at each step the next char of s3 can come from s1 or s2 (if it matches) — branch both ways. That’s O(2m+n) with heavy overlap on (i, j), which is exactly the DP state.

Optimized approach

Require len(s1) + len(s2) == len(s3). Let dp[i][j] = can the first i + j chars of s3 be formed from the first i of s1 and first j of s2. Then dp[i][j] is true if either dp[i−1][j] && s1[i−1] == s3[i+j−1] (take from s1) or dp[i][j−1] && s2[j−1] == s3[i+j−1] (take from s2).

Time O(m·n), Space O(m·n) (reducible to O(n)).

C# solution

C# 
public bool IsInterleave(string s1, string s2, string s3)
{
    int m = s1.Length, n = s2.Length;
    if (m + n != s3.Length) return false;

    var dp = new bool[m + 1, n + 1];
    dp[0, 0] = true;
    for (int i = 0; i <= m; i++)
        for (int j = 0; j <= n; j++)
        {
            if (i > 0 && dp[i - 1, j] && s1[i - 1] == s3[i + j - 1]) dp[i, j] = true;
            if (j > 0 && dp[i, j - 1] && s2[j - 1] == s3[i + j - 1]) dp[i, j] = true;
        }
    return dp[m, n];
}

Complexity

⏱️ Complexity

  • Time: O(m·n).
  • Space: O(m·n), reducible to O(n) with one row.

Pattern & takeaways

🎯 Remember these three

  • State = chars consumed from each source; i + j indexes s3.
  • Reach a cell from above (s1) or left (s2) when the char matches.
  • The length check is a quick necessary precondition.

Interview tip

🗣️ When this comes up

The framing “dp[i][j] = used i of s1 and j of s2” with i+j indexing s3 is the unlock. It’s a reachability grid like Unique Paths, but gated by character matches.
