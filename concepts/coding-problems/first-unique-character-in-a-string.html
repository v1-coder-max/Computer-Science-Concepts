First Unique Character in a String

Which character shows up exactly once, earliest? Count everyone first, then take the first with a tally of one.

#hash-map #counting #strings #easy

The problem

Given a string, return the index of the first character that does not repeat, or -1 if there is none.

📥 Example

"leetcode" → 0 ('l'); "aabb" → -1

Brute force

For each character, scan the rest of the string to see if it repeats — O(n²). Counting all frequencies up front collapses that to two linear passes.

Optimized approach

First pass: tally every character's frequency. Second pass: walk left to right and return the index of the first character whose count is exactly one.

Time O(n), Space O(1).

C# solution

C# 
public int FirstUniqChar(string s)
{
    var count = new int[26];
    foreach (char c in s) count[c - 'a']++;
    for (int i = 0; i < s.Length; i++)
        if (count[s[i] - 'a'] == 1) return i;
    return -1;
}

Complexity

⏱️ Complexity

  • Time: O(n) — two passes over the string.
  • Space: O(1) — a fixed 26-element count.

Pattern & takeaways

🎯 Remember these three

  • Count then scan: two passes beat the O(n²) repeat-search.
  • The second pass must go left-to-right to honor first unique.
  • Order-independent counting plus order-sensitive scanning is a common combo.

Say two passes out loud: one to count, one to find. If asked about a stream where characters arrive over time, mention a queue of candidates plus counts for O(1) amortized lookups.
