House Robber

Rob houses along a street to maximize loot — but tripping two adjacent alarms gets you caught. At each house the choice is simple: take it and skip the neighbor, or skip it.

#dynamic-programming #take-or-skip #o1-space #medium

The problem

Given nums where nums[i] is the money in house i, return the maximum you can rob without robbing two adjacent houses.

📥 Example

[2,7,9,3,1] → 12 (rob houses 0, 2, 4 → 2 + 9 + 1)  ·  [1,2,3,1] → 4

Brute force

Try every valid subset of non-adjacent houses — exponential. But the choice at each house only depends on results just behind it, so the subproblems overlap: DP collapses it to linear time.

Optimized approach

Let dp[i] = the most you can rob from the first i+1 houses. At house i you either skip it (keep dp[i-1]) or rob it (its money plus dp[i-2], since i-1 is now off-limits): dp[i] = max(dp[i-1], nums[i] + dp[i-2]). Each state needs only the previous two answers, so track two rolling variables — O(1) space.

Time O(n), Space O(1).

C# solution

C# — O(1) space DP 
public int Rob(int[] nums)
{
    int prev2 = 0;   // best up to house i-2
    int prev1 = 0;   // best up to house i-1
    foreach (int money in nums)
    {
        int take = money + prev2;            // rob this house
        int curr = Math.Max(prev1, take);    // vs. skip it
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}

Complexity

⏱️ Complexity

  • Time: O(n) — one pass over the houses.
  • Space: O(1) — two rolling variables.

Pattern & takeaways

🎯 Remember these three

  • The take-or-skip recurrence: max(skip, value + best-two-back).
  • State depends only on the last two results → O(1) rolling variables.
  • “Houses in a circle” (House Robber II) = run this twice, excluding the first or last house.

Interview tip

🗣️ When this comes up

State the recurrence in words first — “rob this plus best two back, or skip and keep best so far” — then code the two-variable version. The natural follow-up is the circular street; say you solve it by running the linear version twice.
