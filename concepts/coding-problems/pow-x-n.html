Pow(x, n)

Raise x to the power n without n multiplications. Exponentiation by squaring halves the exponent each step, turning O(n) into O(log n).

#math #divide-and-conquer #fast-exponentiation #medium

The problem

Implement Pow(x, n) — compute x raised to the integer power n (n may be negative). Return a double.

📥 Example

Pow(2.0, 10) → 1024.0  ·  Pow(2.0, -2) → 0.25

Brute force

Multiply x by itself n times — O(n). For large n that’s far too slow, and it does redundant work: x⁸ recomputes everything it already built for x⁴.

Optimized approach

Use the identity xⁿ = (x²)^(n/2) for even n, and x · x^(n-1) for odd n. Each step squares the base and halves the exponent, so you reach n = 0 in about log₂ n steps. For negative powers, compute the positive power and return its reciprocal (and use a long for n to safely negate int.MinValue).

Time O(log n), Space O(1) (iterative).

C# solution

C# — iterative fast power 
public double MyPow(double x, int n)
{
    long e = n;                  // widen so we can negate int.MinValue safely
    if (e < 0) { x = 1 / x; e = -e; }

    double result = 1.0;
    while (e > 0)
    {
        if ((e & 1) == 1) result *= x;  // exponent bit set → multiply in current power
        x *= x;                          // square the base
        e >>= 1;                          // halve the exponent
    }
    return result;
}

Complexity

⏱️ Complexity

  • Time: O(log n) — the exponent halves each iteration.
  • Space: O(1) iterative (O(log n) call stack if recursive).

Pattern & takeaways

🎯 Remember these three

  • Exponentiation by squaring: square the base, halve the exponent → O(log n).
  • The exponent’s binary bits decide which squared powers to multiply in.
  • Handle n < 0 via reciprocal, and widen to long for the int.MinValue edge case.

Interview tip

🗣️ When this comes up

The two things they check: the O(log n) squaring insight, and the edge cases — negative n and n = int.MinValue (which overflows on negation as an int). Call out the long widening; it’s the detail that trips most candidates.
