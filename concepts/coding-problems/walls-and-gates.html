Walls and Gates

Label every empty room with the distance to its nearest gate. Seed BFS from all gates at once and the first arrival is the shortest distance.

#graphs #bfs #grid #medium #multi-source #interview

The problem

A grid has walls (-1), gates (0), and empty rooms (INF). Fill each empty room with the distance to its nearest gate; unreachable rooms stay INF.

📥 Example

A 4×4 grid where each empty room is replaced by its shortest distance to a gate.

Brute force

From each empty room, BFS to the nearest gate — O((m·n)²). Reversing it — BFS from the gates simultaneously — visits each room once at its correct distance.

Optimized approach

Use multi-source BFS: enqueue all gates first (distance 0). Expand level by level; when you reach an empty room (still INF), set its distance to current + 1 and enqueue it. Because all gates start together, the first time a room is dequeued it’s via a shortest path, so no room is updated twice.

Time O(m·n), Space O(m·n).

C# solution

C# 
public void WallsAndGates(int[][] rooms)
{
    const int INF = int.MaxValue;
    int m = rooms.Length, n = rooms[0].Length;
    var q = new Queue<(int r, int c)>();
    for (int r = 0; r < m; r++)
        for (int c = 0; c < n; c++)
            if (rooms[r][c] == 0) q.Enqueue((r, c));      // all gates are sources

    int[][] dirs = { new[]{1,0}, new[]{-1,0}, new[]{0,1}, new[]{0,-1} };
    while (q.Count > 0)
    {
        var (r, c) = q.Dequeue();
        foreach (var d in dirs)
        {
            int nr = r + d[0], nc = c + d[1];
            if (nr >= 0 && nc >= 0 && nr < m && nc < n && rooms[nr][nc] == INF)
            {
                rooms[nr][nc] = rooms[r][c] + 1;          // first reach = shortest
                q.Enqueue((nr, nc));
            }
        }
    }
}

Complexity

⏱️ Complexity

  • Time: O(m·n) — each room enqueued once.
  • Space: O(m·n) for the queue.

Pattern & takeaways

🎯 Remember these three

  • Nearest-source distances → multi-source BFS (seed all sources).
  • INF acts as the “unvisited” marker — only fill rooms still INF.
  • Same template as Rotting Oranges and 01-Matrix.

Interview tip

🗣️ When this comes up

Seeding BFS from all gates at once is the trick — it gives every room its nearest distance in one sweep. Without it you BFS from each room separately. It’s the same multi-source pattern as Rotting Oranges.
