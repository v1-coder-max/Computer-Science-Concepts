Convert BST to Greater Tree

Each node should grow by the sum of everything larger than it. Reverse in-order visits values largest-first, so a single running sum does it.

#trees #bst #dfs #medium

The problem

Given a BST, replace every node's value with its original value plus the sum of all values greater than it.

📥 Example

Largest node keeps its value; the next adds the largest; and so on down to the smallest, which adds everything.

Brute force

Computing each node's greater-sum independently is O(n²). A reverse in-order pass accumulates the sums in O(n).

Optimized approach

Traverse right → node → left (reverse in-order), which visits values from largest to smallest. Keep a running sum; at each node add its value to sum, then overwrite the node with sum.

Time O(n), Space O(h).

C# solution

C# 
public TreeNode ConvertBST(TreeNode root)
{
    int sum = 0;
    void Reverse(TreeNode node)
    {
        if (node == null) return;
        Reverse(node.right);
        sum += node.val;
        node.val = sum;          // value + all greater values
        Reverse(node.left);
    }
    Reverse(root);
    return root;
}

Complexity

⏱️ Complexity

  • Time: O(n) — each node visited once.
  • Space: O(h) — recursion = height.

Pattern & takeaways

🎯 Remember these three

  • Reverse in-order (right, node, left) visits a BST largest-first.
  • A single running sum turns this into one pass.
  • Same trick powers 'BST to greater sum tree' variants.
