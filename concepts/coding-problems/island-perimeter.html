Island Perimeter

Every land cell starts with four edges; each shared edge between two land cells hides two of them. Count the leftover boundary.

#matrix #simulation #easy

The problem

Given a grid with exactly one island (connected 1s), return its perimeter.

📥 Example

A plus-shaped island → perimeter 16

Brute force

You must look at each cell, but you do not need traversal — perimeter is a local property of each land cell and its neighbors.

Optimized approach

For every land cell, add 4 and subtract 1 for each of its land neighbors (a shared edge isn't part of the boundary). Summing over all land cells gives the perimeter without any DFS.

Time O(m·n), Space O(1).

C# solution

C# 
public int IslandPerimeter(int[][] grid)
{
    int rows = grid.Length, cols = grid[0].Length, perim = 0;
    for (int r = 0; r < rows; r++)
        for (int c = 0; c < cols; c++)
            if (grid[r][c] == 1)
            {
                perim += 4;
                if (r > 0 && grid[r - 1][c] == 1) perim -= 2;     // shared top edge
                if (c > 0 && grid[r][c - 1] == 1) perim -= 2;     // shared left edge
            }
    return perim;
}

Complexity

⏱️ Complexity

  • Time: O(m·n) — one scan of the grid.
  • Space: O(1) — constant.

Pattern & takeaways

🎯 Remember these three

  • Perimeter is a local property: 4 minus land neighbors per cell.
  • Counting only top/left neighbors and subtracting 2 avoids double-counting shared edges.
  • No traversal needed — a single scan suffices.
