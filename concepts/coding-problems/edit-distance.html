Edit Distance

Fewest inserts, deletes, or replaces to turn one string into another (Levenshtein). A 2-D DP: equal chars carry the diagonal, else 1 + min of three neighbors.

#dynamic-programming #2d-dp #strings #hard #levenshtein #interview

The problem

Given two strings word1 and word2, return the minimum number of single-character insertions, deletions, or replacements to convert word1 into word2.

📥 Example

"horse" → "ros" → 3  ·  "intention" → "execution" → 5

Brute force

Recursively, at each position either the characters match (advance both) or you pay 1 for an insert, delete, or replace and recurse — an exponential tree of choices. Memoizing on (i, j) collapses it; the table is the bottom-up form.

Optimized approach

dp[i][j] = edits to turn the first i chars of word1 into the first j of word2. If the current chars are equal, dp[i][j] = dp[i−1][j−1] (free). Otherwise it’s 1 + min( dp[i−1][j] delete, dp[i][j−1] insert, dp[i−1][j−1] replace ). Base cases: converting to/from an empty string costs its length.

Time O(m·n), Space O(m·n).

C# solution

C# 
public int MinDistance(string a, string b)
{
    int m = a.Length, n = b.Length;
    var dp = new int[m + 1, n + 1];
    for (int i = 0; i <= m; i++) dp[i, 0] = i;   // delete all
    for (int j = 0; j <= n; j++) dp[0, j] = j;   // insert all

    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++)
            dp[i, j] = a[i - 1] == b[j - 1]
                ? dp[i - 1, j - 1]
                : 1 + Math.Min(dp[i - 1, j - 1],                 // replace
                      Math.Min(dp[i - 1, j], dp[i, j - 1]));     // delete / insert
    return dp[m, n];
}

Complexity

⏱️ Complexity

  • Time: O(m·n).
  • Space: O(m·n), reducible to O(min(m,n)).

Pattern & takeaways

🎯 Remember these three

  • Three operations map to three neighbors: delete, insert, replace.
  • Equal characters cost nothing — inherit the diagonal.
  • Base row/column are 0..n (turn an empty string into a prefix).

Interview tip

🗣️ When this comes up

Map each of the three edits to a neighbor cell — that mapping is the whole solution. Edit distance and LCS share the same grid shape; pointing that out signals you see the family, not just the instance.
