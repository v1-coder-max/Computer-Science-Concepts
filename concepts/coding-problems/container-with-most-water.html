Container With Most Water

Pick two vertical lines that hold the most water. Start wide and walk inward, always moving the shorter wall — that’s the only move that can ever help.

#two-pointers #arrays #greedy #medium

The problem

Given heights height[i], each a vertical line at position i, pick two lines that with the x-axis form a container holding the most water. Return that maximum area.

📥 Example

[1,8,6,2,5,4,8,3,7] → 49 (lines at index 1 and 8: min(8,7) × 7)

Brute force

Try every pair of lines and compute min(height[i], height[j]) × (j - i) — O(n²). It works but re-examines pairs that can never beat what we’ve seen. A two-pointer sweep gets it in one pass.

Optimized approach

Put one pointer at each end. The area is min(left, right) × width. To improve, you must beat the current area despite the width shrinking — so you need a taller wall. Moving the taller wall in can never raise the limiting height; only moving the shorter wall has a chance. So compute the area, then advance the shorter side. Track the max as you converge.

Time O(n), Space O(1).

C# solution

C# 
public int MaxArea(int[] height)
{
    int left = 0, right = height.Length - 1, best = 0;
    while (left < right)
    {
        int h = Math.Min(height[left], height[right]);
        best = Math.Max(best, h * (right - left));   // area with current walls
        if (height[left] < height[right]) left++;    // move the shorter wall
        else right--;
    }
    return best;
}

Complexity

⏱️ Complexity

  • Time: O(n) — the two pointers meet after n steps.
  • Space: O(1) — two indices.

Pattern & takeaways

🎯 Remember these three

  • Opposite-end two pointers shine when the answer depends on a width and a min/max of the ends.
  • Move the side that limits you — moving the taller wall can’t help.
  • The greedy move is safe because the discarded pairs are all dominated.

Interview tip

🗣️ When this comes up

The whole interview is “why is moving the shorter wall safe?” Answer it: width shrinks every step, so the only path to a bigger area is a taller limiting wall. Saying that proves the greedy choice isn’t a guess.
