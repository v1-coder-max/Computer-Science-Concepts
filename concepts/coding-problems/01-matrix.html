01 Matrix

Seed the BFS queue with every 0 at once. The wave expands outward, stamping each cell with its distance to the nearest 0 on first arrival.

#graphs #bfs #matrix #medium

The problem

Given a binary matrix, return a matrix where each cell holds its distance (in 4-directional steps) to the nearest 0.

📥 Example

[[0,0,0],[0,1,0],[1,1,1]] → [[0,0,0],[0,1,0],[1,2,1]]

Brute force

A BFS from each 1 to its nearest 0 is O((m·n)²). A single multi-source BFS from all zeros does it in O(m·n).

Optimized approach

Initialize distances: 0 for zero-cells, infinity for ones. Enqueue every 0. Run BFS; when expanding to a neighbor with a larger tentative distance, set it to current + 1 and enqueue it. First arrival is the shortest distance.

Time O(m·n), Space O(m·n).

C# solution

C# 
public int[][] UpdateMatrix(int[][] mat)
{
    int rows = mat.Length, cols = mat[0].Length;
    var dist = new int[rows][];
    var q = new Queue<(int, int)>();
    for (int r = 0; r < rows; r++)
    {
        dist[r] = new int[cols];
        for (int c = 0; c < cols; c++)
        {
            if (mat[r][c] == 0) q.Enqueue((r, c));
            else dist[r][c] = int.MaxValue;
        }
    }
    int[][] dirs = { new[]{1,0}, new[]{-1,0}, new[]{0,1}, new[]{0,-1} };
    while (q.Count > 0)
    {
        var (r, c) = q.Dequeue();
        foreach (var d in dirs)
        {
            int nr = r + d[0], nc = c + d[1];
            if (nr >= 0 && nc >= 0 && nr < rows && nc < cols
                && dist[nr][nc] > dist[r][c] + 1)
            {
                dist[nr][nc] = dist[r][c] + 1;
                q.Enqueue((nr, nc));
            }
        }
    }
    return dist;
}

Complexity

⏱️ Complexity

  • Time: O(m·n) — each cell enqueued at most once.
  • Space: O(m·n) — the distance grid and queue.

Pattern & takeaways

🎯 Remember these three

  • Multi-source BFS: seed the queue with every 0 at once.
  • First arrival at a cell is its shortest distance — BFS guarantees it.
  • Same pattern as Rotting Oranges and Walls and Gates.

Interview tip

🗣️ When this comes up

The leap from many single-source BFS runs to one multi-source BFS is the whole optimization. Seeding all zeros and expanding one ring at a time is the reusable template.
