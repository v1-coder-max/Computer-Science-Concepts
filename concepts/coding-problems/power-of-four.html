Power of Four

A power of four is a power of two whose lone set bit sits at an even position — test both with bit masks.

#bit-manipulation #math #easy

The problem

Given an integer n, return whether it is a power of four (1, 4, 16, 64, …).

📥 Example

16 → true; 8 → false (power of two, wrong position)

Brute force

Looping division by 4 is O(log n). Bit tricks make it O(1).

Optimized approach

n must be a power of two (n > 0 && (n & (n−1)) == 0) and its single set bit must be at an even position. Test the latter with a mask of even-position bits: n & 0x55555555 != 0.

Time O(1), Space O(1).

C# solution

C# 
public bool IsPowerOfFour(int n)
{
    return n > 0
        && (n & (n - 1)) == 0          // power of two
        && (n & 0x55555555) != 0;      // bit at an even position
}

Complexity

⏱️ Complexity

  • Time: O(1) — a couple of bitwise operations.
  • Space: O(1) — constant.

Pattern & takeaways

🎯 Remember these three

  • Power of four ⇔ power of two with its bit at an even position.
  • n & (n−1) == 0 tests the single-bit condition.
  • The mask 0x55555555 selects even bit positions (1, 4, 16, …).
