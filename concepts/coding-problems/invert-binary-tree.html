Invert Binary Tree

Mirror a tree left-to-right. Swap each node’s two children and recurse — three lines that show you understand tree recursion.

#trees #recursion #dfs #easy

The problem

Given the root of a binary tree, invert it (swap left and right subtrees everywhere) and return the root.

📥 Example

[4,2,7,1,3,6,9] → [4,7,2,9,6,3,1] — every level reads right-to-left.

Brute force

There’s no real “slow” version — any correct approach visits every node once (O(n)). The question is really testing whether you can express the recursion (or its iterative equivalent) cleanly.

Optimized approach

Inverting a tree = swap the root’s left and right children, then invert each subtree. That’s a direct recursion: base case is null; otherwise swap, recurse left, recurse right. Equivalently, do a BFS/DFS with a queue or stack and swap children as you pop each node.

Time O(n), Space O(h) (recursion/stack depth = tree height).

C# solution

C# 
public class TreeNode { public int val; public TreeNode left, right; }

public TreeNode InvertTree(TreeNode root)
{
    if (root == null) return null;
    (root.left, root.right) = (InvertTree(root.right), InvertTree(root.left));
    return root;
}

// Iterative (BFS) variant:
public TreeNode InvertIterative(TreeNode root)
{
    if (root == null) return null;
    var q = new Queue<TreeNode>();
    q.Enqueue(root);
    while (q.Count > 0)
    {
        var n = q.Dequeue();
        (n.left, n.right) = (n.right, n.left);   // swap
        if (n.left  != null) q.Enqueue(n.left);
        if (n.right != null) q.Enqueue(n.right);
    }
    return root;
}

Complexity

⏱️ Complexity

  • Time: O(n) — every node is visited once.
  • Space: O(h) recursion depth (O(n) worst case for a skewed tree).

Pattern & takeaways

🎯 Remember these three

  • Most tree problems = do something at the node, recurse into both children.
  • The base case (null) is what stops the recursion — always state it first.
  • Any recursive tree walk has an iterative twin using an explicit stack or queue.

Interview tip
