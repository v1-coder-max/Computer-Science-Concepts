Graph Valid Tree

When is a graph a tree? Exactly when it has n−1 edges, is fully connected, and has no cycle — and the first two imply the third.

#graphs #union-find #medium #tree #connectivity #interview

The problem

Given n nodes and a list of undirected edges, return true if they form a valid tree (connected and acyclic).

📥 Example

n=5, edges=[[0,1],[0,2],[0,3],[1,4]] → true  ·  with an extra edge forming a cycle → false

Brute force

DFS to detect a cycle (tracking the parent to ignore the edge you came from) and separately verify all nodes are visited — correct in O(n + e). Union-Find packages both checks neatly.

Optimized approach

A tree on n nodes has exactly n − 1 edges, so first check edges.Length == n − 1 — fail fast otherwise. Then ensure connectivity with Union-Find: union every edge; if any edge connects two nodes already in the same set, there’s a cycle → false. With exactly n − 1 edges and no cycle, the graph is automatically connected, so it’s a valid tree.

Time ~O(n·α(n)), Space O(n).

C# solution

C# 
public bool ValidTree(int n, int[][] edges)
{
    if (edges.Length != n - 1) return false;     // tree must have exactly n-1 edges

    var parent = new int[n];
    for (int i = 0; i < n; i++) parent[i] = i;
    int Find(int x) => parent[x] == x ? x : (parent[x] = Find(parent[x]));

    foreach (var e in edges)
    {
        int a = Find(e[0]), b = Find(e[1]);
        if (a == b) return false;                // cycle
        parent[a] = b;
    }
    return true;                                 // n-1 edges + acyclic ⇒ connected
}

Complexity

⏱️ Complexity

  • Time: ~O(n·α(n)) with path compression.
  • Space: O(n).

Pattern & takeaways

🎯 Remember these three

  • Tree ⇔ connected + acyclic ⇔ n−1 edges + acyclic.
  • The edge-count check is an O(1) fail-fast.
  • Union-Find catches the cycle; the count guarantees connectivity.

Interview tip

🗣️ When this comes up

Lead with the characterization: n−1 edges plus no cycle implies connected. That lets you skip a separate connectivity pass. Union-Find or parent-tracking DFS both work — know why the edge count matters.
