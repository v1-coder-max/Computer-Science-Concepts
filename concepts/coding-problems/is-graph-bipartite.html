Is Graph Bipartite

Can the graph be 2-colored so no edge joins same colors? Color it greedily with BFS/DFS — a conflict means it isn’t bipartite.

#graphs #bfs #coloring #medium #bipartite #interview

The problem

Given an undirected graph as an adjacency list, return true if it is bipartite (its nodes can be split into two sets with every edge crossing between them).

📥 Example

[[1,3],[0,2],[1,3],[0,2]] → true  ·  a triangle → false

Brute force

Trying all 2-set partitions is exponential. A graph is bipartite iff it has no odd-length cycle, which a single 2-coloring traversal detects directly in O(V+E).

Optimized approach

Maintain a color array (0 = uncolored). For each uncolored node, start a BFS/DFS, assigning it one color and each neighbor the opposite color. If you ever reach a neighbor already colored the same as the current node, there’s an odd cycle → not bipartite. Repeat across all components (the graph may be disconnected).

Time O(V + E), Space O(V).

C# solution

C# 
public bool IsBipartite(int[][] graph)
{
    int n = graph.Length;
    var color = new int[n];          // 0 = uncolored, 1 / -1 = the two sides

    for (int start = 0; start < n; start++)
    {
        if (color[start] != 0) continue;
        var q = new Queue<int>();
        q.Enqueue(start); color[start] = 1;
        while (q.Count > 0)
        {
            int node = q.Dequeue();
            foreach (int next in graph[node])
            {
                if (color[next] == 0) { color[next] = -color[node]; q.Enqueue(next); }
                else if (color[next] == color[node]) return false;   // conflict
            }
        }
    }
    return true;
}

Complexity

⏱️ Complexity

  • Time: O(V + E).
  • Space: O(V) for colors and queue.

Pattern & takeaways

🎯 Remember these three

  • Bipartite ⇔ 2-colorable ⇔ no odd cycle.
  • Assign opposite colors across each edge; a same-color edge fails.
  • Loop over all nodes to cover disconnected components.

Interview tip

🗣️ When this comes up

State the equivalence (bipartite ⇔ 2-colorable ⇔ no odd cycle) and color greedily. Remember to iterate all components — forgetting disconnected pieces is the common miss.
