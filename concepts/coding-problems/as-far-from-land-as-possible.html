As Far from Land as Possible

Seed the BFS with every land cell at once; the wave expands ring by ring, and the last water cell it reaches is the farthest.

#graphs #bfs #matrix #medium

The problem

Given a grid of 0s (water) and 1s (land), return the maximum Manhattan distance from any water cell to its nearest land, or −1 if the grid is all land or all water.

📥 Example

[[1,0,1],[0,0,0],[1,0,1]] → 2 (the center)

Brute force

BFS from each water cell to its nearest land is O((m·n)²). A single multi-source BFS from all land is O(m·n).

Optimized approach

Enqueue every land cell with distance 0. BFS outward, stamping each newly reached water cell with the current distance. The last distance assigned (the deepest ring) is the answer; −1 if no expansion happened.

Time O(m·n), Space O(m·n).

C# solution

C# 
public int MaxDistance(int[][] grid)
{
    int n = grid.Length;
    var q = new Queue<(int r, int c)>();
    for (int r = 0; r < n; r++)
        for (int c = 0; c < n; c++)
            if (grid[r][c] == 1) q.Enqueue((r, c));
    if (q.Count == 0 || q.Count == n * n) return -1;
    int[][] dirs = { new[]{1,0}, new[]{-1,0}, new[]{0,1}, new[]{0,-1} };
    int dist = -1;
    while (q.Count > 0)
    {
        dist++;
        for (int k = q.Count; k > 0; k--)
        {
            var (r, c) = q.Dequeue();
            foreach (var d in dirs)
            {
                int nr = r + d[0], nc = c + d[1];
                if (nr >= 0 && nc >= 0 && nr < n && nc < n && grid[nr][nc] == 0)
                {
                    grid[nr][nc] = 1;        // mark visited
                    q.Enqueue((nr, nc));
                }
            }
        }
    }
    return dist;
}

Complexity

⏱️ Complexity

  • Time: O(m·n) — each cell enqueued once.
  • Space: O(m·n) — the BFS queue.

Pattern & takeaways

🎯 Remember these three

  • Multi-source BFS from all land cells expands distance uniformly.
  • The last ring reached gives the maximum distance.
  • All-land or all-water grids return −1.
