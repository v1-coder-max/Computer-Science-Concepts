N-th Tribonacci Number

Like Fibonacci, but each term sums the previous three. Three rolling variables compute it in one pass.

#dynamic-programming #math #easy

The problem

The Tribonacci sequence is T0 = 0, T1 = T2 = 1, and T(n) = T(n−1) + T(n−2) + T(n−3). Return T(n).

📥 Example

n = 4 → 4 (0, 1, 1, 2, 4)

Brute force

Naive recursion recomputes subproblems exponentially. Iterating with rolling variables is O(n), O(1) space.

Optimized approach

Keep the last three values and slide forward, each step summing them into the next. Handle the small base cases (n < 3) directly.

Time O(n), Space O(1).

C# solution

C# 
public int Tribonacci(int n)
{
    if (n == 0) return 0;
    if (n <= 2) return 1;
    int a = 0, b = 1, c = 1;
    for (int i = 3; i <= n; i++)
    {
        int next = a + b + c;
        a = b; b = c; c = next;
    }
    return c;
}

Complexity

⏱️ Complexity

  • Time: O(n) — one pass.
  • Space: O(1) — three rolling values.

Pattern & takeaways

🎯 Remember these three

  • Each term is the sum of the previous three — slide three variables.
  • Handle n < 3 base cases up front.
  • Same memoize-or-roll lesson as Fibonacci and Climbing Stairs.
