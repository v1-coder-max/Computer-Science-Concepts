Maximum Depth of Binary Tree

How tall is the tree? The height of a node is 1 + the taller of its two subtrees. That single recurrence is the template for half of all tree problems.

#trees #recursion #dfs #easy

The problem

Given the root of a binary tree, return its maximum depth — the number of nodes along the longest path from the root down to a leaf.

📥 Example

[3,9,20,null,null,15,7] → 3  ·  [] → 0

Brute force

There’s no wasteful version worth writing — you must look at every node (O(n)). The exercise is to express the height recurrence correctly and handle the empty-tree base case.

Optimized approach

Define depth recursively: an empty tree has depth 0; otherwise the depth is 1 + max(depth(left), depth(right)). The recursion naturally explores every path and bubbles the maximum back up. The iterative form is a BFS that counts how many levels the queue processes.

Time O(n), Space O(h) (call-stack depth = height).

C# solution

C# 
public int MaxDepth(TreeNode root)
{
    if (root == null) return 0;                       // empty subtree
    return 1 + Math.Max(MaxDepth(root.left), MaxDepth(root.right));
}

// Iterative BFS: count levels.
public int MaxDepthBfs(TreeNode root)
{
    if (root == null) return 0;
    var q = new Queue<TreeNode>();
    q.Enqueue(root);
    int depth = 0;
    while (q.Count > 0)
    {
        int size = q.Count;                           // one full level
        for (int i = 0; i < size; i++)
        {
            var n = q.Dequeue();
            if (n.left  != null) q.Enqueue(n.left);
            if (n.right != null) q.Enqueue(n.right);
        }
        depth++;
    }
    return depth;
}

Complexity

⏱️ Complexity

  • Time: O(n) — visit every node once.
  • Space: O(h) recursion (O(n) for a skewed tree); BFS uses O(width).

Pattern & takeaways

🎯 Remember these three

  • The recurrence 1 + max(left, right) is the canonical “aggregate up the tree” pattern.
  • Empty tree → 0 is the base case that makes leaves return 1.
  • Swap max for min (with a leaf guard) to get minimum depth.
