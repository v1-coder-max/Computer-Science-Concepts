Car Fleet

Cars can’t pass; a faster car catching a slower one joins its fleet at the slower speed. Sort by position and count arrival times that aren’t swallowed ahead.

#stack #sorting #medium #monotonic #simulation #interview

The problem

Cars at given positions head to a target at given speeds; a faster car catching a slower one forms a fleet moving at the slower speed. Return the number of fleets that arrive.

📥 Example

target=12, position=[10,8,0,5,3], speed=[2,4,1,1,3] → 3

Brute force

Simulating positions over time until everything merges is fiddly and slow. Reasoning about arrival times in order of starting position turns it into a single O(n log n) scan.

Optimized approach

Each car’s time to the target is (target − position) / speed. Sort cars by start position descending (closest to the target first) and walk through them, keeping the slowest arrival time so far (the front of the current fleet). If a car’s arrival time is greater than that front time, it can’t catch up — it starts a new fleet and becomes the new front; otherwise it merges into the fleet ahead. Count the new fronts. (The non-increasing fronts behave like a monotonic stack.)

Time O(n log n), Space O(n).

C# solution

C# 
public int CarFleet(int target, int[] position, int[] speed)
{
    int n = position.Length;
    var cars = new (int pos, double time)[n];
    for (int i = 0; i < n; i++)
        cars[i] = (position[i], (double)(target - position[i]) / speed[i]);

    Array.Sort(cars, (a, b) => b.pos - a.pos);   // closest to target first

    int fleets = 0;
    double front = 0;
    foreach (var car in cars)
        if (car.time > front)                    // can't catch the car ahead
        {
            fleets++;
            front = car.time;                    // new fleet front
        }
    return fleets;
}

Complexity

⏱️ Complexity

  • Time: O(n log n) — sort dominates.
  • Space: O(n) for the cars array.

Pattern & takeaways

🎯 Remember these three

  • Reason about arrival times, not moving positions.
  • Sort by position toward the target; a slower front absorbs faster followers.
  • A new fleet starts whenever a car’s time exceeds the running front time.
