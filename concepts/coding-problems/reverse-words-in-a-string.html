Reverse Words in a String

Keep each word intact but flip their order, and tidy up any stray whitespace along the way.

#strings #two-pointers #parsing #medium

The problem

Given a string of words separated by spaces, return the words in reverse order with a single space between them and no leading or trailing spaces.

📥 Example

"the sky is blue" → "blue is sky the"

Brute force

Reach for the language: string.Split on whitespace, drop empties, reverse the list, and join with a single space. Clean and O(n), though it allocates a word list.

Optimized approach

For O(1) extra space (mutating a char buffer), reverse the entire string, then reverse each individual word back. The double reversal lands every word in the right spot with its letters in order.

Time O(n), Space O(n).

C# solution

C# 
public string ReverseWords(string s)
{
    var words = s.Split(' ', StringSplitOptions.RemoveEmptyEntries);
    Array.Reverse(words);
    return string.Join(' ', words);
}

Complexity

⏱️ Complexity

  • Time: O(n) — one pass to split, one to join.
  • Space: O(n) — the array of words.

Pattern & takeaways

🎯 Remember these three

  • Split → reverse → join is the readable answer; the double-reverse trick is the O(1)-space one.
  • Watch the edge cases: leading, trailing, and repeated spaces.
  • RemoveEmptyEntries handles the messy whitespace for free.

Interview tip

🗣️ When this comes up

Ask whether you may use library splits or must mutate a buffer in place. If in-place, describe the reverse-all-then-reverse-each-word trick; it is the detail interviewers are listening for.
