Longest Palindromic Substring

Find the longest palindrome inside a string. Every palindrome has a center — so expand outward from each center and keep the widest.

#strings #two-pointers #palindrome #medium #expand-center #interview

The problem

Given a string s, return the longest substring that is a palindrome.

📥 Example

"babad" → "bab" (or "aba")  ·  "cbbd" → "bb"

The idea

A palindrome is symmetric around a center. There are 2n − 1 possible centers — each character (odd-length palindromes) and each gap between characters (even-length). From each center, expand two pointers outward while the characters match; the widest expansion across all centers is the answer. No DP table needed, and it’s O(1) space.

Time O(n²), Space O(1).

C# solution

C# 
public string LongestPalindrome(string s)
{
    if (s.Length < 2) return s;
    int start = 0, maxLen = 1;

    for (int center = 0; center < s.Length; center++)
    {
        Expand(center, center);       // odd-length
        Expand(center, center + 1);   // even-length
    }
    return s.Substring(start, maxLen);

    void Expand(int l, int r)
    {
        while (l >= 0 && r < s.Length && s[l] == s[r]) { l--; r++; }
        int len = r - l - 1;          // window after the last good match
        if (len > maxLen) { start = l + 1; maxLen = len; }
    }
}

Complexity

⏱️ Complexity

  • Time: O(n²) — n centers, O(n) expansion each.
  • Space: O(1) — no DP table.

Pattern & takeaways

🎯 Remember these three

  • Every palindrome has a center; there are 2n − 1 of them.
  • Handle odd and even centers separately.
  • Expand-around-center beats the O(n²)-space DP; Manacher’s gives O(n).

Interview tip

🗣️ When this comes up

Expand-around-center is the expected O(n²)/O(1) answer — remember both center types. If asked for O(n), name Manacher’s algorithm (you rarely need to code it, but knowing it exists matters).
