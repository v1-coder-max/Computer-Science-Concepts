Find the Difference

Every shared character cancels under XOR, leaving only the one that was added.

#bit-manipulation #hash-map #easy

The problem

String t is string s with one extra character added and then shuffled. Return the added character.

📥 Example

s = "abcd", t = "abcde" → 'e'

Brute force

Counting characters in both strings and finding the surplus works in O(n) time, O(1) space. XOR is even cleaner.

Optimized approach

XOR together the char codes of every character in both s and t. Each character present in both cancels (x ^ x = 0), so the result is the code of the added character.

Time O(n), Space O(1).

C# solution

C# 
public char FindTheDifference(string s, string t)
{
    int x = 0;
    foreach (char c in s) x ^= c;
    foreach (char c in t) x ^= c;
    return (char)x;
}

Complexity

⏱️ Complexity

  • Time: O(n) — one pass over each string.
  • Space: O(1) — a single accumulator.

Pattern & takeaways

🎯 Remember these three

  • XOR cancels every shared character, leaving the added one.
  • Summing char codes and subtracting also works.
  • It's the Single Number trick applied across two strings.
