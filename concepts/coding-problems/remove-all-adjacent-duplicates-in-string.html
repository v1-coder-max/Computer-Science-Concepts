Remove All Adjacent Duplicates In String

A stack lets pairs cancel as they form: if the next character equals the top, they annihilate; otherwise it waits on the stack.

#stack #strings #easy

The problem

Repeatedly remove two adjacent equal characters until none remain, and return the final string.

📥 Example

"abbaca" → "ca"

Brute force

Scanning and deleting pairs repeatedly is O(n²) in the worst case. A stack resolves every cancellation in one pass.

Optimized approach

Scan the string with a stack. If the current character equals the top, pop it (they cancel); otherwise push it. The remaining stack, read bottom to top, is the answer.

Time O(n), Space O(n).

C# solution

C# 
public string RemoveDuplicates(string s)
{
    var stack = new Stack<char>();
    foreach (char c in s)
    {
        if (stack.Count > 0 && stack.Peek() == c) stack.Pop();
        else stack.Push(c);
    }
    var arr = stack.ToArray();
    Array.Reverse(arr);
    return new string(arr);
}

Complexity

⏱️ Complexity

  • Time: O(n) — each character pushed/popped once.
  • Space: O(n) — the stack.

Pattern & takeaways

🎯 Remember these three

  • A stack collapses adjacent duplicates in one linear pass.
  • Equal-to-top ⇒ pop (cancel); else push.
  • Reverse the stack contents to rebuild the result string.
