Ransom Note

Cut letters out of a magazine to spell a note. It works exactly when the magazine has at least as many of each letter as the note needs.

#hash-map #counting #strings #easy

The problem

Given strings ransomNote and magazine, return true if the note can be constructed using each letter of the magazine at most once.

📥 Example

note = "aa", magazine = "aab" → true

Brute force

For each note character, scan the magazine for an unused copy and cross it off — O(n·m). A frequency count removes the inner scan entirely.

Optimized approach

Tally the magazine's letters once. Then walk the note, decrementing the matching count; if any needed letter hits zero with more still required, the note can't be built.

Time O(n), Space O(1).

C# solution

C# 
public bool CanConstruct(string ransomNote, string magazine)
{
    var count = new int[26];
    foreach (char c in magazine) count[c - 'a']++;
    foreach (char c in ransomNote)
        if (--count[c - 'a'] < 0) return false;
    return true;
}

Complexity

⏱️ Complexity

  • Time: O(n) — two linear passes.
  • Space: O(1) — a fixed 26-element count.

Pattern & takeaways

🎯 Remember these three

  • A frequency count turns a nested scan into two linear passes.
  • Decrement while consuming; a count below zero means a shortfall.
  • Lowercase-only inputs let you use a 26-element array instead of a dictionary.
