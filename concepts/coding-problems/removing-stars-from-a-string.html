Removing Stars From a String

Treat the string like typing: a letter pushes, a '*' is a backspace that pops the most recent letter.

#stack #strings #easy

The problem

Each '*' removes the closest non-star character to its left (and itself). Return the string after all stars are applied.

📥 Example

"leet**cod*e" → "lecoe"

Brute force

Repeatedly finding and deleting around each star is O(n²). A stack does it in one pass.

Optimized approach

Scan the string with a stack (a StringBuilder works). Push each letter; on a '*', pop the top. The remaining characters, in order, are the answer.

Time O(n), Space O(n).

C# solution

C# 
public string RemoveStars(string s)
{
    var sb = new StringBuilder();
    foreach (char c in s)
    {
        if (c == '*') sb.Remove(sb.Length - 1, 1);
        else sb.Append(c);
    }
    return sb.ToString();
}

Complexity

⏱️ Complexity

  • Time: O(n) — each character handled once.
  • Space: O(n) — the output buffer.

Pattern & takeaways

🎯 Remember these three

  • A '*' is a backspace — model it with a stack.
  • A StringBuilder used as a stack avoids a final reverse.
  • Same pattern as Backspace String Compare and adjacent-duplicate removal.
