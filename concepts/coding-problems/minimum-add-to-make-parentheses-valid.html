Minimum Add to Make Parentheses Valid

Sweep once tracking how many '(' are open; a ')' with nothing open needs an added '(', and any '(' left over each need a ')'.

#stack #greedy #strings #easy

The problem

Return the minimum number of parentheses to insert so a string of '(' and ')' becomes valid.

📥 Example

"())(" → 2

Brute force

Searching insertion points is unnecessary. A single balance counter pinpoints every unmatched bracket.

Optimized approach

Track an open balance. A '(' increments it; a ')' decrements it, but if it's already 0, that ')' is unmatched — count an addition instead. At the end, the remaining open count is the number of ')' still needed.

Time O(n), Space O(1).

C# solution

C# 
public int MinAddToMakeValid(string s)
{
    int open = 0, add = 0;
    foreach (char c in s)
    {
        if (c == '(') open++;
        else if (open > 0) open--;
        else add++;            // unmatched ')'
    }
    return add + open;          // plus leftover '('
}

Complexity

⏱️ Complexity

  • Time: O(n) — single pass.
  • Space: O(1) — two counters.

Pattern & takeaways

🎯 Remember these three

  • A running balance counts unmatched ')' on the fly.
  • Leftover open count = the ')' you still need to add.
  • No stack required — a single integer balance suffices.

Interview tip

🗣️ When this comes up

Reduce the stack to a counter: an unmatched ')' needs a '(', and leftover '(' need ')'. Summing both gives the answer in O(1) space.
