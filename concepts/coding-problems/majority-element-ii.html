Majority Element II

All values appearing more than n/3 times — and there are at most two. Boyer-Moore voting with two candidates finds them in O(1) space.

#arrays #boyer-moore #medium #voting #o1 #interview

The problem

Given an array nums, return all elements that appear more than ⌊n/3⌋ times. There can be at most two such elements.

📥 Example

[3,2,3] → [3]  ·  [1,1,1,3,3,2,2,2] → [1,2]

Brute force

A hash map of counts is O(n) time but O(n) space. Since more-than-n/3 elements number at most two, the voting algorithm generalizes to track two candidates in O(1) space.

Optimized approach

Keep two candidates and two counters. For each value: if it matches a candidate, bump that counter; else if a counter is 0, adopt the value as that candidate (count 1); else decrement both counters. This survives the two majority elements. Because the candidates are only guaranteed when a >n/3 element exists, do a second pass to verify each candidate actually exceeds n/3.

Time O(n), Space O(1).

C# solution

C# 
public IList<int> MajorityElement(int[] nums)
{
    int c1 = 0, c2 = 1, n1 = 0, n2 = 0;        // two candidates, two counts
    foreach (int v in nums)
    {
        if (v == c1) n1++;
        else if (v == c2) n2++;
        else if (n1 == 0) { c1 = v; n1 = 1; }
        else if (n2 == 0) { c2 = v; n2 = 1; }
        else { n1--; n2--; }
    }

    var result = new List<int>();
    foreach (int c in new[] { c1, c2 })
        if (nums.Count(x => x == c) > nums.Length / 3) result.Add(c);
    return result.Distinct().ToList();
}

Complexity

⏱️ Complexity

  • Time: O(n) — two passes.
  • Space: O(1) — two candidates and counters.

Pattern & takeaways

🎯 Remember these three

  • At most two elements can exceed n/3 → two voting slots.
  • A verification pass is required (candidates aren’t guaranteed).
  • Generalizes the single-candidate Boyer-Moore to k candidates for >n/k.

Interview tip

🗣️ When this comes up

The leap from Majority Element is “at most two candidates, so track two,” plus the mandatory verification pass. Explaining why exactly two slots suffice is the depth they probe.
