Find Largest Value in Each Tree Row

One number per level: the biggest. A level-by-level BFS makes it a simple per-row max.

#trees #bfs #medium

The problem

Given a binary tree, return a list containing the largest value in each row (level), top to bottom.

📥 Example

Tree [1,3,2,5,3,null,9] → [1, 3, 9]

Brute force

A DFS that records depth and updates a per-depth maximum works equally well; BFS just matches the row structure of the output.

Optimized approach

BFS level by level. For each level, scan its nodes and track the maximum, then append it to the result before moving to the next level.

Time O(n), Space O(w).

C# solution

C# 
public IList<int> LargestValues(TreeNode root)
{
    var res = new List<int>();
    if (root == null) return res;
    var q = new Queue<TreeNode>();
    q.Enqueue(root);
    while (q.Count > 0)
    {
        int size = q.Count, best = int.MinValue;
        for (int i = 0; i < size; i++)
        {
            var node = q.Dequeue();
            best = Math.Max(best, node.val);
            if (node.left != null) q.Enqueue(node.left);
            if (node.right != null) q.Enqueue(node.right);
        }
        res.Add(best);
    }
    return res;
}

Complexity

⏱️ Complexity

  • Time: O(n) — each node enqueued once.
  • Space: O(w) — w = widest level.

Pattern & takeaways

🎯 Remember these three

  • Capture the level size, then take the max over that many nodes.
  • Initialise the per-level best to int.MinValue for negatives.
  • Reuses the standard level-order traversal skeleton.
